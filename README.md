# Silent Auction

This Python program allows multiple users to enter their bids in a silent auction. 
At the end, it announces the highest bidder and their bid amount. 

## How to run
1. Run the script: `python silent_auction.py`
2. Enter your name and bid amount.
3. Repeat for all bidders.
4. Type 'no' when there are no more bidders to see the winner.
